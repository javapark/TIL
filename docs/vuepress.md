# Vuepress 로 TIL 만들기
## Getting Started
```
# package.json
yarn init

# Install VuePress locally
yarn add -D vuepress

# Create first document
mkdir docs && echo '# Hello VuePress' > docs/README.md

# Serve the documentation site in the local server
```
